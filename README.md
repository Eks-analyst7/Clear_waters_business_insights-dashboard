# Clear Waters Communications Business Insights Dashboard

## Project Overview

This project was developed as part of my Data Analytics training and focuses on analyzing customer application performance for Clear Waters Communications.

The objective was to transform raw business data into meaningful insights that support decision-making and improve operational efficiency.

Using Power BI, SQL, and MySQL, I designed an interactive dashboard that provides a comprehensive view of application trends, approval rates, withdrawal patterns, and store performance.
