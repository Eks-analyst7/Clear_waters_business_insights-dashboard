# Clear Waters Communications Business Insights Dashboard

## Introduction

The Clear Waters Communications Business Insights Dashboard is a Power BI project developed to analyze customer application performance, operational efficiency, and business KPIs.

The dashboard transforms raw business data into actionable insights through interactive visualizations and performance tracking metrics, enabling data-driven decision-making.

---

## Objectives

* Analyze customer application outcomes.
* Monitor approval, decline, and withdrawal rates.
* Evaluate analyst and store performance.
* Track business KPIs using interactive dashboards.
* Support management decision-making through data visualization.

---

## Tools Used

* Power BI
* SQL
* MySQL
* Microsoft Excel

---

## Dashboard Pages

### Home Dashboard

Provides an overview of business performance through key metrics and KPI cards.

### Credit Analytics Team

Analyzes analyst performance, application trends, approval rates, and withdrawal reasons.

### Store Insights

Evaluates store-level performance and application processing efficiency.

---

## Key Performance Indicators

* Total Applications
* Approval Rate
* Decline Rate
* Withdrawal Rate
* Applications Within SLA
* Average Handling Time

---

## Project Workflow

1. Collected and prepared business data.
2. Built and managed the database using MySQL.

4. Created data models and DAX measures.
5. Designed interactive Power BI dashboards.
6. Generated business insights for decision-making.
